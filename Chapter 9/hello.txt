9. What happens if an existing file is opened in write mode?
If an existing file is opened in write mode ('w'), its contents are erased and the file pointer is placed at the beginning of the file, allowing new data to be written from the start

10. hat is the difference between the read() and readlines() methods?
The read() method reads the entire content of a file as a single string or bytes object, depending on the file mode 
whereas readlines() reads the entire file and returns a list where each element is a line from the file.
